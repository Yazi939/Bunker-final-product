Если серверы не запускаются автоматически:

  1. Закройте это окно
  2. Запустите MongoDB: mongod --dbpath=./data/db
  3. Запустите API сервер: cd server && npm start
  4. Запустите React: npm run start
  5. Запустите Electron: npm run electron:dev

Или просто запустите файл run-app.bat
