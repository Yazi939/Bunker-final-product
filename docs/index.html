Основные возможности

Учет операций

Добавление и управление операциями покупки и продажи топлива

Аналитика

Анализ прибыльности и маржинальности операций с топливом

Экспорт в Excel

Экспорт данных и отчетов в формат Excel для дальнейшей обработки

Статистика

Подробная статистика по видам топлива с визуализацией

Системные требования

  • Windows: Windows 10 или 11, 2 ГБ ОЗУ, 500 МБ свободного места
  • macOS: macOS 10.13 или новее, 2 ГБ ОЗУ, 500 МБ свободного места

Инструкции по установке

Для Windows:

  1. Скачайте файл установщика BunkerBoats-Setup.exe
  2. Запустите файл для установки приложения
  3. Следуйте инструкциям установщика
  4. Если Node.js не обнаружен, установщик предложит автоматически скачать и установить его
  5. После установки запустите приложение через созданный ярлык

Для macOS:

  1. Скачайте DMG-файл приложения
  2. Откройте DMG-файл и перетащите приложение в папку Applications
  3. Запустите приложение из Launchpad или папки Applications
